# Printful to do app

To-do list app made for the Printful summer school test using PHP 7.4.7, MySQL, TailwindCSS.

## Installation

### 1. Add the included .dump file to your MySQL database.

### 2. Compile the CSS

```bash
$ npm run build:css
```

### 3. Launch the PHP server

```bash
$ php -S localhost:8000
```

### 4. Go to localhost:8000 and use the project
